Hermes Moneymanager for Android

Introduction

Hermes Moneymanager for Android is a port of the desktop personal finance application Hermes Moneymanager, currently available for Windows, Mac OSX and Linux. The Android versiown mirrors the best features of the desktop version, while adding mobility and the database synchronization.

In Hermes Moneymanager project, we strive to develop a good personal finance software that we use ourselves. We also do this for fun and the love of programming. We are a non-profit, informal organization although we do accept donations and use them towards the hosting fees. Any leftovers for the Android version will also go towards acquiring a device for the app testing.

Features

All the feature requests can be tracked at our issue tracker on .

Highlights

  • Opening databases by using storage access framework, allowing you to pick your favourite storage provider, whether cloud or local.
  • Transaction management
  • Multiple accounts and account types, currencies, and languages.
  • Account, payee, category, currency management
  • Budgets (view only, currently)
  • Investment watchlists
  • Asset Allocation
  • Storage card Import/Export
  • Security - application lock/unlock with pin code
  • Recurring transactions
  • Reports and Charts

Planned

  • Database encryption support (prototype ready!)
  • Attachments

Testing - Pre-Release Versions

Your assistance with testing the pre-release (Beta) versions of the app is highly appreciated. This way you can directly help the project and ensure that the stable releases are bulletproof.

Should you want to try out the latest version of Hermes Moneymanager for Android, you can access it in two ways:

  • By installing Beta version as a separate app from Google Play Store.
  • From our Releases page on GitHub. Simply download and install the latest version manually. You need to enable installing from unknown sources in Android settings first.

Previously we used a Beta Testing Program but this option is not as convenient as having the Beta version as a separate app. With Beta as a separate app you always have the stable version as the fallback option should you identify any issues in the unstable version.
